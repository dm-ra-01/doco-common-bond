# About-Us Documentation Audit

**Date:** 2026-03-11\
**Scope:** `documentation/common-bond` — `docs/about-us/` section\
**Auditor:** Ryan Ammendolea\
**Standard:** Internal documentation completeness; audience: advisors, future
hires, partner relationships

---

## Executive Summary

The Docusaurus internal corporate site hosts rich strategy, compliance, and
technical documentation, but has no section that provides a cohesive,
human-readable narrative about _who Common Bond is_. The `docs/about-us/`
directory was created but left empty. This audit identifies **1 documentation
gap finding (INFO-01)** rated 🟡 Low — no security or process risk exists, but
the missing section represents a material gap for external onboarding and
internal coherence. Implementation will create 5 content pages, a sidebar entry,
and a navbar item.

## 1. About-Us Documentation Gap

### 1.1 Content Completeness

**Strengths:**

- The `docs/strategy/strategy-vision/` section contains deep, well-structured
  documentation on the founder, vision, core values, competitive strengths, and
  why-now analysis — this provides a rich source of truth to draw from.
- `strategy/brand/common-bond.md` and `operations/eos/vto.md` collectively
  document the company identity and operational cadence.
- The `strategy/strategy-vision/founder.md` contains a comprehensive
  professional profile with education, clinical career, engineering career,
  service history, and personal vision statement.

**Gaps:**

- **INFO-01** `docs/about-us/` — The directory was explicitly created
  (`mkdir'd`) but contains zero files. No index, no sub-pages. A visitor
  navigating to the About Us section encounters a 404 or an empty generated
  index with no content. There is no single document that synthesises _who
  Common Bond is_, _who Ryan is_ (in a narrative rather than CV style), _what we
  do_, _where we currently are_, and _what we are aiming for_.

- **INFO-02** `sidebars.ts` — No `aboutUsSidebar` entry exists. The `about-us`
  directory will not appear in Docusaurus navigation without a corresponding
  autogenerated sidebar definition.

- **INFO-03** `docusaurus.config.ts` — No navbar item for About Us. The section
  would be unreachable from the top-level navigation.

## 2. Cross-Cutting Observations

1. **Narrative-Strategy Gap:** The site contains significant
   structured/quantitative planning materials (V/TO, revenue projections, SWOT,
   scope matrices) but no _narrative_ layer that tells the Common Bond story in
   prose. The about-us section will be the first place a new advisor, future
   employee, or partner relationship visits.

2. **Founder Page Duplication Risk:** `strategy/strategy-vision/founder.md` is a
   comprehensive CV. The new `about-us/founder.md` should be narrative-first and
   link _to_ the CV page rather than duplicating it. The audit recommends
   keeping these as distinct documents with clear cross-links.

3. **Legal Entity Clarity:** The `who-we-are` page provides an opportunity to
   clearly articulate the MyJMO Pty Ltd / Common Bond Pty Ltd / Ammendolea
   Family Trust (ABN: 50 152 085 287) structure — this is currently undocumented
   in the public-facing parts of the site, which is a minor compliance gap (ISO
   27001 Clause 5.3, organisational roles and responsibilities).
