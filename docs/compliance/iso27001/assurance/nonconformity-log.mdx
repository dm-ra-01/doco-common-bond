# Non-Conformity Log

This log records all non-conformities identified through internal audits,
management reviews, incident investigations, and external assessments.
Each non-conformity has a linked corrective action (CA) tracked in the
[Corrective Actions register](./corrective-actions).

:::note[ISO 27001 Clause 10.2]

Clause 10.2 requires that the organisation react to non-conformities, take action
to control and correct them, deal with the consequences, evaluate the need for action
to eliminate root causes, and implement any action needed. Non-conformities and
corrective actions must be retained as documented information.

:::
