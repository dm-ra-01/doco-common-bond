# Annex A Information Security Controls

Annex A of ISO/IEC 27001:2022 contains 93 information security controls grouped into 4 themes.

## Control Themes

| Theme | Description | Controls |
|-------|-------------|----------|
| **[Organizational](./organizational)** | Policies, rules, and processes. | 37 Controls |
| **[People](./people-physical#6-people-controls)** | HR, awareness, and training. | 8 Controls |
| **[Physical](./people-physical#7-physical-controls)** | Sites, equipment, and facilities. | 14 Controls |
| **[Technological](./technological)** | IT systems, networks, and software. | 34 Controls |
