# ISMS Roles & Responsibilities

## 1. Overview
Information security is everyone's responsibility. As a small team, we wear multiple hats, but distinct accountability is crucial for ISO 27001 compliance.

## 2. Key Roles

### Top Management (Founder / CEO)
**Accountability:** Ultimate responsibility for the ISMS.
**Responsibilities:**
- Approving the Information Security Policy.
- Ensuring resources are available for security initiatives.
- Conducting Management Reviews.
- Accepting residual risks.

### Information Security Manager (ISM) / CTO
**Accountability:** Operational design and maintenance of the ISMS.
**Responsibilities:**
- Leading the risk assessment process.
- Monitoring security metrics and incidents.
- Coordinating internal audits.
- Ensuring technical controls (e.g., Supabase RLS, Cloudflare WAF) are effective.

### All Employees & Contractors
**Responsibility:** "If you see something, say something."
**Responsibilities:**
- Adhering to the Acceptable Use Policy.
- Reporting security incidents or weaknesses immediately.
- Completing security awareness training.
- Keeping devices secure (updates, disk encryption, strong passwords).
