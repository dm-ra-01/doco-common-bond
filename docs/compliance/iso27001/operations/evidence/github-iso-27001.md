# GitHub ISO 27001 Certification

Evidence of GitHub's ISO 27001:2022 certification, as required for supplier security assessment (Annex A 5.19-5.22).

## Certificate Details

| Field | Value |
| :--- | :--- |
| **Supplier** | GitHub, Inc. |
| **Standard** | ISO/IEC 27001:2022 |
| **Certificate Number** | 2024-020901 |
| **Certification Body** | Coalfire Certification, Inc. |
| **Original Registration** | May 6, 2022 |
| **Current Issuance** | May 1, 2025 |
| **Expiration Date** | May 5, 2028 |

## Scope

The ISMS supporting the operations underlying the following GitHub.com offerings:
- **GitHub Enterprise Cloud (GHEC)** and GitHub Enterprise Cloud Data Resident
- **GitHub Advanced Security (GHAS)**: Code Scanning, Dependabot, Secret Scanning
- **GitHub Actions**: Machine Management Service
- **GitHub Copilot**: Business, Enterprise
