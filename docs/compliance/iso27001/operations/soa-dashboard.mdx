Use the filters below to explore controls by **applicability**, **implementation status**, and **theme**. The full static SoA is at [operations/soa](/docs/compliance/iso27001/operations/soa).

## Implementation Status Definitions

| Status | Meaning |
|--------|---------|
| **Implemented** | Control is fully operational with evidence |
| **Partial** | Control partially implemented — remediation in progress |
| **Planned** | Control scheduled for implementation |
| **Not Started** | Control not yet actioned |
| **Not Applicable** | Control excluded with documented justification |
