## Register Types

| Type | Description |
|:---|:---|
| **ISMS** | Registers mandated by ISO 27001:2022 |
| **Engineering** | Registers maintained by the engineering governance programme |
| **Corporate** | Registers maintained at the corporate entity level |

## Status Definitions

| Status | Meaning |
|:---|:---|
| **Active** | Register is current, maintained, and reviewed on schedule |
| **In Progress** | Register is being migrated or developed |
| **Planned** | Register creation scheduled but not yet commenced |
| **Needs Review** | Register is overdue for its scheduled review |
