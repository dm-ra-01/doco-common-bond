# 🔔 Governance Review Dashboard

Outstanding review alerts for all active governance records in the [`supabase-common-bond`](https://github.com/dm-ra-01/supabase-common-bond) database.

Review alerts are generated daily at **08:00 AEST** by the `func_send_review_alert()` pg_cron job (REC-04 / ALERT-01). A record appears here when its `review_due_date` is within 30 days of today.

## Review Cadence

| Register | Review Cadence |
|----------|---------------|
| Risk Register | Annual / on material change |
| NC Log / Corrective Actions | Continuous |
| Asset Register | Annual |
| Supplier Register | Annual |
| SoA Controls | Annual |
| Training Records | Per onboarding event |
| Standards Register | Annual / on policy change |
| Audit Registry | Per audit |
