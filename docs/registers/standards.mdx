## Standard Types

| Type | Description |
|:---|:---|
| **Policy** | High-level mandatory intent statements (ISO 27001 Clause 5.2) |
| **Procedure** | Step-by-step operational instructions for implementing policies |
| **Standard** | Technical or engineering standards governing implementation |
| **Framework** | External reference frameworks (e.g., ISO 27001, NIST, OWASP) |

## Status Definitions

| Status | Meaning |
|:---|:---|
| **Active** | Current and in force |
| **Draft** | Under development — not yet approved |
| **Under Review** | Scheduled for formal review — current version still in force |
| **Retired** | Superseded or withdrawn — retained for historical reference |
